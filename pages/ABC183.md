```
def solve(N, K, dist):
    from itertools import permutations
    ret = 0
    for order in permutations(range(1, N)):
        d = 0
        prev = 0
        for x in order:
            d += dist[prev][x]
            prev = x
        d += dist[prev][0]
        if d == K:
            ret += 1
    return ret
```

- 公式解説
