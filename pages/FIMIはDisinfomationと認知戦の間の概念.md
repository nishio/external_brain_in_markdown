## 1. Disinformation / Misinformation / Malinformation
これは 「情報そのものの性質」 による分類です。
_
| 用語 | 意図 | 内容 |
| -- | -- | -- |
| Misinformation | 悪意なし | 誤情報・勘違い・誤解 |
| Disinformation | 悪意あり | 意図的な虚偽情報 |
| Malinformation | 悪意あり | 事実だが、文脈を歪めて害を与える情報 |
ポイント：
- 個人・メディア・善意の拡散者も含まれる
- 国家関与を前提としない

## 2. FIMI（Foreign Information Manipulation and Interference）
FIMI は EU・NATO 文脈で確立された概念で、「外国アクターによる、組織的・継続的な情報操作・干渉」を指します。

FIMIの特徴
- 外国アクター（国家・準国家組織）が関与
- 戦略目的（民主主義弱体化、分断、信頼低下など）
- 手段はDisinformation / Misinformation / Malinformationを含むが、それに限定されない
    - 含まれるものの例：
        - ボットネット・偽アカウント運用
        - ナラティブ操作
        - 増幅（amplification）
        - タイミング操作
        - クロスプラットフォーム展開
    - 👉 Disinformation / Misinformation / Malinformation は「使われうる材料」
- 👉 FIMI は「作戦・オペレーション単位」

## 3. 認知戦（Cognitive Warfare）
認知戦は、さらに上位の概念です。
「人間の認知領域（信念・感情・判断・意思決定）を戦場とする戦争」

認知戦の射程
- 情報操作だけに限らない
- 心理・文化・制度・社会的信頼まで含む
- 真偽の問題を超える
例：
- 「何が真実か」ではなく「誰を信じるか」「考える気力を失わせるか」
- 分断の固定化
- 無力感・シニシズムの醸成
👉 FIMIは、認知戦の代表的な実行手段の一つ
