
herokuではアクセスがないと30分でサーバが終了する。
そこでherokuにデプロイしたボットに、終了時に「sleeping...」と送信する機能をつけた
python

```
import atexit
@atexit.register
def exit():
    requests.post("<URL>", json=dict(text="(heroku)sleeping..."))
```


しかし、このメッセージがherokuに送られて、眠るとすぐに起こされるw
