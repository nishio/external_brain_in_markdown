# Ver.1 [[Bitcoin]]
- 概要: Bitcoinは、[[分散型台帳技術]]であるブロックチェーンを用いた最初の[[暗号通貨]]です。Bitcoinのブロックチェーンは、[[トランザクションの記録]]と確認に利用され、第三者機関を介さずに[[P2P]]での金銭の送受信を可能にします。
- 差分: この段階では、ブロックチェーン技術は主に金銭取引の記録に限定されていました。

# Ver.2 [[Ethereum]]
- 概要: Ethereumは、Bitcoinの基本的なブロックチェーン機能に加え、[[スマートコントラクト]]を実行できるプラットフォームです。これにより、単なる通貨の送受信を超え、自動的に実行される[[契約]]やアプリケーションの開発が可能になりました。
- 差分: Ethereumの主な進化点は、スマートコントラクトの導入です。これにより、[[分散型アプリケーション]]（[[DApps]]）の開発が可能になり、ブロックチェーン技術の応用範囲が大きく広がりました。

# Ver.3 [[Designated Verifier Signature]]
- 概要: Designated Verifier Signatureは、特定の検証者のみが署名の正当性を検証できるデジタル署名技術です。この概念をブロックチェーンと結びつけると、ブロックチェーン上でのプライベートなトランザクションやメッセージの交換を可能にするセキュリティ技術として捉えることができます。
- 差分: [[プライバシー]]とセキュリティに重点を置き、特定の受信者のみがトランザクションの内容や正当性を確認できるようにすることです。これは、ブロックチェーン技術を用いた通信の[[秘匿性]]を高めることに貢献します。

# Ver. 3.1
- Ver.3.1は、Designated Verifier Signature（Ver.3）の概念に基づくが、さらなる改良や応用を加えたものと考えられます。具体的には、以下のような進化が考えられます：
- 改良点: プライバシー保護技術のさらなる強化、例えば、[[ゼロ知識証明]]を組み合わせることで、トランザクションの存在や正当性を証明しつつ、その内容を完全に隠蔽すること。
- 応用例: [[分散型IDシステム]]、[[投票システム]]、[[秘密のデータ交換]]など、プライバシーを必要とするあらゆる分野での応用。
