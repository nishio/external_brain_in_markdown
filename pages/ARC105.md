```
def solve(XS):
    S = sum(XS)
    if S % 2 == 1:
        return False
    goal = S // 2
    for i in range(16):
        s = 0
        for j in range(4):
            if i & 1:
                s += XS[j]
            i >>= 1
        if s == goal:
            return True
    return False


def main():
    # parse input
    XS = list(map(int, input().split()))
    if solve(XS):
        print("Yes")
    else:
        print("No")
```

    - ビットで部分集合全体を探索してる
- 公式解説
    - [[ソートしても一般性を失わない]]
    - ソートされてる場合に等号条件を満たすパターンは2つしかない
    - DとC、DとBが同じグループに入るとダメだから
        - D + C > B + A (D > B, C > A)
        - D + B > C + A (D > C, B > A)

            - 要するにmod x
        - イコールの時→終了

            - これ[[ユークリッドの互除法]]だな

```
def solve(XS):
    from math import gcd
    from functools import reduce
    return reduce(gcd, XS)
```
