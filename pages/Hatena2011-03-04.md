*1299229916*解決: src refspec master does not match any. failed to push some refs to...
GitHubに手元のチョコっとしたコードを入れてみようと思ってcreate reposして、手元でgit initしてgit addしてpushしたらこけた
>||
$ git push -u origin master
error: src refspec master does not match any.
error: failed to push some refs to 'git@github.com:nishio/googletrans_ja_en_ja.git'
||<
コミットしてないじゃん自分！
>||
$ git commit -m "first commit"
