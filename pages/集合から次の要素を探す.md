
ある値aが与えられて、集合からa < xである最小のxを見つけたい

集合を値域が0/1のフェニック木で表現する。
- `sum(a)`: $\sum_{0 \le i < a} X_i$
- `bisect(s)`: minimum x s.t. sum(x) >= s
この時
python

```
def find_next(self, pos):
    s = self.sum(pos + 1) + 1
    return self.bisect(s) - 1
```
