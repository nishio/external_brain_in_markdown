- 公式解説とまったく同じ
- 想定質問「どうやって思いつくんですか？」
    - 小さい具体的な値について解いてください。X=5、D=2とかで。
    - Xの絶対値がDより大きい間は減らす方向に動く
    - 小さくなると原点を挟んで振動する
        - 周期は2なので残り回数を2で割った余りを見る
python

```
def solve(X, K, D):
    X = abs(X)
    a = X // D
    if a >= K:
        return X - K * D
    r = X % D
    K -= a
    if K % 2 == 0:
        return r
    else:
        return D - r
```
