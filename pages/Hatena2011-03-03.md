*1299114301*e-Tax日記(続編)
さて、区役所でロックの解除とパスワードの再設定をしてきましたよ。再設定しないでもロックだけ解除できたっぽいけどわからないで両方やって同じパスワードを設定しなおしてしまった。まあいい。待ち時間がなかったので10分くらいでした。

もう一度確認
- カードリーダのドライバインストール済み
- ルート証明書インストール済み
- 利用者クライアントソフトインストール済み

よし、次からパスワード入力ダイアログが出たらとりあえず証明書のを入れるぞ！Macのパスワードを間違えてもMacはロックされないからな！

利用者クライアントソフトを起動、「自分の証明書」を表示。パスワードを入れる。読めた！

eTaxトップページに戻って前回の途中経過保存ファイルをロードする。「次へ」を押す。…
前回と同じエラーで死亡。エラーコードはeTaxのヘルプに載ってないんだよなこれ。コピペできないエラーコードを目でコピーしてGoogleで検索する。

<a href='http://blog.livedoor.jp/sdfg158-dbungutecho/archives /1564620.html'>デジタル文具手帳 β : E-TaxのMacからの電子申請でエラーコード：AF0020-698が表示されて困ったときは</a>
>>
- サファリを64ビットモードで動かしている場合はアプリケーションアイコンから「情報を見る」で情報画面を開き、「32ビットモードで開く」をオンにする
- 私は念のため Mac OS X 10.6.6上で、「Java for Mac OS X 10.6 アップデート 3」を入れた
<<

おお、次回はこれを確認だな！今日のところはタイムアップだ。現在のところ消費時間は2時間。
