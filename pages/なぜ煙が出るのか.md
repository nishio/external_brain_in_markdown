炭火に薪を追加しても燃えない、煙が出る、科学的にはどういう現象？<img src='https://scrapbox.io/api/pages/nishio/nishio/icon' alt='nishio.icon' height="19.5"/>

- 「低温の[[燻焼]]（[[smoldering]]）」が起きていて、揮発分は出るのに燃える条件（温度・酸素）が足りず、未燃の粒子やタールが煙になって出ています。
- 吸熱で温度が落ちる
    - 薪が冷たい／湿っていると、まず乾燥（水の蒸発潜熱 ≈2.3 MJ/kg）で大量に熱を奪う。
    - 次に[[熱分解]]（[[乾留]]）が始まるが、これも基本は吸熱。炭床の温度が下がり、炎が起きにくい。
- 酸素が足りない
    - 炭は酸素不足だと CO を多く出す（C + 1/2 O₂ → CO）。この CO も温度が足りないと燃えきらない。
- 煙の正体（未燃物と微粒子）
    - 低温の熱分解で出たタール滴、未燃の炭化水素、すす（微小炭素）、水蒸気、有機酸などが、火炎に入らず凝縮エアロゾル＝白／灰色の煙として見える。
- いつ「炎」になるか
    - 十分な温度（薪表面が着火温度域に達する）と十分な酸素（対流・ドラフト）が同時に満たされると、揮発分に[[拡散火炎]]がつき、煙は急に減ります。

拡散火炎とは<img src='https://scrapbox.io/api/pages/nishio/nishio/icon' alt='nishio.icon' height="19.5"/>
<img src='https://scrapbox.io/api/pages/nishio/GPT5/icon' alt='GPT5.icon' height="19.5"/>
- 拡散火炎（diffusion flame／非予混合火炎）は、燃料と酸素（空気）が最初は混ざっておらず、界面で拡散しながら混ざった“その場”で燃える火です。ろうそく、空気穴を閉じたバーナー、薪や炭の上で立つ炎、油のプール火災などが典型例。
- 拡散火炎：混合が遅く、混合（拡散・渦混合）が燃焼速度を律速。界面の当量（ストイキ）面に薄い反応層（フレームシート）ができる。
    - 燃料リッチ核（ガス発生源の近傍）—O₂が乏しい
    - 反応層（フレームシート）—混合比が当量になる薄膜で主反応が進む
    - 外側の酸化帯—CO/H₂/すす等の二次酸化
- 反応層の局所温度は、当量近傍なら理論火炎温度 ~ 1700–2100 ℃（燃料による）に近づくが、実際は放射・希釈で低下。
- 薪・炭の現場では、ガスが当量面で≳800–1000 ℃まで上がり、十分に酸素が巻き込まれると拡散火炎が安定して立ち、煙が減る。混合不足や冷却で600–800 ℃域だとタール酸化が追いつかず白/黄煙になりやすい。

木を炭火に乗せたときの温度帯
    - 〜100 ℃：乾燥（自由水の除去）
        - H₂O(l) → H₂O(g)（蒸発潜熱 2.26 MJ/kg）
    - 150–200 ℃：脱酢酸など（へミセルロース初期分解、酢酸やメタノール発生）
        - ここで酢酸ができるから酸っぱい匂いになるんだな
        - [[木酢液]]ってこれのこと？
    - 200–260 ℃：へミセルロース熱分解（CO₂、酢酸、軽質ガス）
    - 240–350 ℃：セルロース熱分解（レボグルコサン＝タール主成分、CO、CO₂、H₂、CH₄）
        - ここで[[タール]]が発生している
        - これが完全に分解しないとあちこちに付着して匂いのもとになる
    - 280–500 ℃：リグニン熱分解（フェノール類・芳香族タール）
    - 300–400 ℃：木材表面の自己着火温度帯（条件依存）
    - 400–600 ℃：チャー（炭）酸化が優勢、赤熱開始
    - ≥600–800 ℃：揮発分・CO/H₂の二次燃焼が進み炎が安定（タール分解も進む）
    - ≥700–900+ ℃：タール熱分解・すす酸化が進んで煙が減少
        - 目視目安：暗赤550–650 ℃、さくら色730–800 ℃、オレンジ930–1000 ℃

代表的な化学反応（25 ℃基準・標準反応熱）
乾燥・熱分解（吸熱・発煙源）
- 乾燥：H₂O(l) → H₂O(g)　+40.7 kJ/mol（= 2.26 MJ/kg）
- 木材熱分解（模式）：(C₆H₁₀O₅)ₙ → C_(s) + タール + CO + CO₂ + H₂ + CH₄ + …（吸熱、生成物は条件依存）
表面（固体）での炭/木炭の反応
- 完全酸化：C + O₂ → CO₂　−393.5 kJ/mol
- 不完全酸化：C + ½O₂ → CO　−110.5 kJ/mol（酸素不足で増える）
- ブドワール反応：C + CO₂ → 2 CO　+172 kJ/mol（強い吸熱→床温低下）
- 水性ガス反応：C + H₂O(g) → CO + H₂　+131 kJ/mol（吸熱→床温低下）
気相での二次燃焼（炎側）
- COの酸化：CO + ½O₂ → CO₂　−283 kJ/mol（**約≥600 ℃**で速くなる）
- H₂の酸化：H₂ + ½O₂ → H₂O(g)　−242 kJ/mol
- メタンの酸化：CH₄ + 2O₂ → CO₂ + 2H₂O(g)　−802 kJ/mol

自己着火・燃焼の温度目安（条件で変動）
- 木材表面：約300–400 ℃
- CO：約609 ℃
- H₂：約500–585 ℃
- CH₄：約537 ℃
- タール蒸気：組成依存だが**概ね≥300–500 ℃で着火しやすく、≥700–800 ℃**で分解・燃えやすい

なぜ「薪を足すと煙る」のか（数値感）
- 含水20%の薪1 kgだと、水0.2 kgを100 ℃まで昇温（≈63 kJ）+ 蒸発（≈451 kJ）で≈0.51 MJを奪う。
さらに木質部（≈0.8 kg）を25→300 ℃へ上げるだけで≈0.33 MJ（比熱≈1.5 kJ/kgK仮定）。
合計**≈0.8 MJ以上の吸熱で炭床温が600–800 ℃域から落ちやすい**。
- 温度が**<600–700 ℃に落ちるとCO二次燃焼が鈍化**、揮発分が炎に入れず、タールが凝縮して白/青煙が目立つ。
- さらにCO₂/水蒸気が炭と反応（ブドワール／水性ガス、いずれも吸熱）→追い打ちで冷える。
要するに、吸熱（乾燥・熱分解・ガス化）で温度が落ちる＋一時的な酸素不足で、
「発生ガスの気相燃焼に必要な ≥600–800 ℃」に届かず→未燃のタール/すす＝煙が出ます。

- Q1: 酢酸の酸っぱい匂いをさせない方法は？
    - 酸っぱい匂いの主犯は、木が 150–260℃あたりで出す「酢酸などの含酸素化合物（＋水蒸気）」が、炎で“二次燃焼”しきらずに鼻へ届くこと
- Q2: タールが飛散しないようにするには？
    - タールは「低温熱分解で出た重い有機物」が、**冷えて液滴化（エアロゾル化）**して飛びます。


    - 乾いた薪を使う（最優先）
        - 含水が高いほど 100℃近辺の乾燥に熱を吸われて、酢酸が出る温度帯で“煙だけ出て燃えない”になりがち。
    - 樹種・部位を選ぶ
        - 皮（樹皮）や腐りかけ、針葉樹のヤニ多めは煙・刺激臭が出やすい。調理前は特に避ける。
    - 最初は「細割り」中心
        - 太い薪は内部が低温熱分解になりやすく、酸っぱい煙を長く出しがち。
    - “炎が立ってる層”を必ず通す配置にする
        - 酢酸やタール蒸気は、炎の中を通って 800℃級に触れると一気に減りやすい。
        - 逆に、炭の上にベタッと寝かせて無炎の還元雰囲気で乾留させると酸臭が出やすい。
