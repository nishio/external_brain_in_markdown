システム2
- 3つ目の状態が導入された
- ![image](https://gyazo.com/e3ba9a9fda5723bb1ac8e996c8c98150/thumb/1000)
:
| -0- | -0- | -1- | -1- | -2- | -20 | -21 | -22 |
| -- | -- | -- | -- | -- | -- | -- | -- | -- | -- | -- | -- | -- |
|  A |  B |  A |  B |  A |  B |  B |  B |
| -1-  | -2-  | -2- | -2-  | -1-  | -00  | -12  | -11 |
|  __B | A__ | A__ | __B | A__ | __A | __B | __B |
| -0- | -0- | -0- | -1- | -1- | -10 | -11 | -12 | -2- | -20 | -21 | -22 | -2- |
|  A |  B |  C |  A |  B |  C |  C |  C |  A |  B |  B |  B |  C |
| -1-  | -2- | -2-  | -2-  | -2-  | -0-  | -1-  | -1-  | -1- | -0-  | -1-  | -1-  | -2- |
| __B | A__ | A__ | A__ | __B | __A | __C | __C | A__ | __A | __C | __C | __B |
- システム2でCに入る代わりにBに入って右に進んだあと2なら1、1なら2に書き換えると、システム1に一致する
:
| -21 | -22 |
| -- | -- |
| B |  B |
| -12  | -11 |
| __B | __B |
| -21 | -22 |
|  B |  B |
| -1-  | -1-  |
| __C | __C |
- システム2で状態Cである時の振る舞いは、システム1でBである時の振る舞いの1と2を入れ替えたものと同じ
:
| -0- | -20 | -21 | -22 | -1- |
| -- | -- | -- | -- | -- |
|  B |  B |  B |  B |  B |
| -2-  | -00  | -12  | -11 | -2-  |
| A__ | __A | __B | __B | __B |
| -0- | -10 | -11 | -12 | -2- |
|  C |  C |  C |  C |  C |
| -2-  | -0-  | -1-  | -1-  | -2- |
| A__ | __A | __C | __C | __B |
- なので、システム1のテープをそのままシステム2に渡せば、状態Cに入る時の書き換えの差が、状態Cでの判断の差とちょうど打ち消しあって、問題なく動く
- 逆も同様だが、始状態がCの時はテープの現在位置を書き換える必要がある

![image](https://gyazo.com/7f79ad3c39327d35b96c65b418867a02/thumb/1000)

このシステム3を使って、0で区切られた1と2の一塊によって任意の非負整数を表現できる
- 二進法とはちょっと違う

システム5を使って二色サイクリックタグシステムを実現する

[[two-colour cyclic tag system]]
[Cyclic Tag System -- from Wolfram MathWorld](https://mathworld.wolfram.com/CyclicTagSystem.html)
- タグシステムとは、n個のルールを順に適用したあと最初のルールに戻るもの
- サイクリックタグシステムはルールが「先頭が1である時、末尾にパターンを追加する。先頭の値によらず取り除く」の形をしているもの

二色サイクリックタグシステムに帰着できれば、[[2-タグシステム]]に帰着できて、チューリングマシンに帰着できる
