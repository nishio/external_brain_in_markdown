というわけで括弧は全部なくなった。

>|python|
import sys

exec "N = " + sys.argv[1]
mem = [[1, 1]]
n = 1
while n < N:
    n += 1
    mem += [[n, mem[-1][1] * n]]

for i, f in mem:
    print "%s! = %%s" % i % f
||<

一般に関数をカッコなしで呼び出す方法があれば「もう括弧ゴルフとか飽きたし」っていえるなーと思って少し考えてみたが、タプルがカッコなしで作成できることを利用すれば簡単だった。

>|python|
>>> x = 1, 2
>>> x
(1, 2)
>>> `x`
'(1, 2)'
>>> exec "y = max" + `x`
>>> y
2
||<

引数を取らない関数を呼ぶときはスライシングで空のタプルを作ればいい。

>|python|
>>> x[:0]
()
||<
