方法2: ブランチ全体を[[bundle]]で移す
- bundleファイルを作成
    - 元リポジトリで移行するブランチのbundleを作成します
    - `$ git bundle create branch.bundle ブランチ名`
- 移行先リポジトリでbundleを取り込む
    - 移行先リポジトリで以下のコマンドを実行
    - `$ git pull /path/to/branch.bundle ブランチ名`

※履歴もそのまま取り込むので、移行先のベースと合わない場合は調整が必要です。
