    - 「AはBに勝つ」という関係が遷移的でない
:

```
A = (0, 0, 4, 4, 4, 4),
B = (3, 3, 3, 3, 3, 3),
C = (2, 2, 2, 2, 6, 6),
D = (1, 1, 1, 5, 5, 5).
```

    - 期待値の計算と違って「勝つ確率」の計算の際には「小さく勝っても、大きく負けても、勝ち負けとしては1回」という非線形関数が掛かるのでそれを利用して「期待値が同じだけど勝率の高いダイス」を作ることができる。
