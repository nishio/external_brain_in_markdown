2021-10-17時点の結論
- 不可視で当たり判定のあるブロックがbarrier(バリア)という名前で用意されている
- 見た目と当たり判定の両立を目指そうとするをやめた方がいい
    - 当たり判定が必要なところにはバリアを置く
    - 見た目は不可視の額縁にいれたアイテムで実現する
    - …という割り切りをするのが脳の負担が少ないのではないか


以下最初の考察
![image](https://gyazo.com/c66c416a20492040e4e4096eabd58682/thumb/1000)
フルブロックと上つきハーフブロック(スラブ)の上にはアイテムフレーム(額縁)が置ける
感圧版と下つきハーフブロックは、アイテムフレームが浮く
カーペットは感圧版と違って隙間なく並ぶが、アイテムフレームが置けない

:
|  | 床フレーム | 下スラブ | ブロック | 階段 | 上スラブ | 感圧版 | カーペット |
| -- | -- | -- | -- | -- | -- | -- | -- |
| フレーム置ける | x | x | o | o | o | x | x |
| 45度回転 | o | x | x | x | x | x | x |
| 90度回転 | o | x | x | o | x | x | x |
| 照明 | o | x | x | * | o | o | o |
| 座れる | x | o | o | o | o | o | o |

椅子に関しては不可視のブロック/上スラブにフレームで見た目をかぶせればよい
- 上にさらに物を積むことがないのでフレームが置けないことは問題にならない
![image](https://gyazo.com/a29d6c1aac5a6defd31823d6ce8cbdc4/thumb/1000)

机に関して、1ブロックだと村人が登ってしまうのも問題
- 高さ2に浮かせて配置するか
結合して広い机を作ろうとすると問題

フェンスを使えば高さ1.5だから村人は乗らない
- 結合するモデルなのでそれを使えば広い机も作れるかも
- 十字の時に隣の形状がわからないからフルの幅である必要はある
- 角を丸めることはできる
- [[Minecraft Fence Model]]
![image](https://gyazo.com/054cdc33bd8a8c349b68cb7cbaca7dc3/thumb/1000)
- 残念ながらこれは普通には額縁が置けなかった
- fixed属性の額縁をsummonすればいいかも

昔はフェンスの上に置いたトラップドアの上にものが配置できたようだが今はできなさそう

上面が高さ2.0であるようなハーフブロックで四角い机を作るのが一番安定する方法
複雑な形状を作ろうとした場合、ハーフブロックが向きを持たないことが問題になる
