
- 量子アニーリングマシンに入力するためにイジングモデルを作りたい
- つまり変数の2次までの項で表現したい
- a and bという論理式は、a, bを{0, 1}の変数とするなら a * b でよい
- では a and b and c はどうやって表現すればよいか？
- 単純に a * b * c にすると三次の項ができてしまうぞ？という問題

- 解法1
    - [イジング模型への変換 – Quantum Computing Information Site](https://quantum.fixstars.com/introduction_to_quantum_computer/quantum_annealing/programming/conversion/)
    - 新しい変数dを導入する方法
- 僕の解法1
    - a and b and c が成立するということは、すべてが1だということ
    - (a + b + c - 3)^2 は全てが1の時だけ0になり、それ以外は正の値になる
:

```

[1, 0, 0] 0 4
```

    - 最後のケースだけ右端の値(a + b + c - 3)^2が0になっている
- 僕の解法2
    - 解法1ではたまたま「よい置き換え方法」を人間が思いつく必要があった
    - これを機械にやらせられないか？
    - 全入力パターンと2次までの掛け合わせを特徴量とし、正解データと合わせてロジスティック回帰で機械学習
    - この程度のサイズなら学習は一瞬(20変数ぐらいまでなら現実的な速度で計算できると思う)
    - 結果、scoreが1.0になるので、線形分離であることがわかる
        - 線形分離でない場合(変数を追加しなければいけない場合)はスコアが1.0にならないのでわかるはず
        - その場合にどうやって問題を解けばいいかはまだわからない
