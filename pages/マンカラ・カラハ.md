- 運の要素のない2人完全情報ゲーム

双方が相手陣地に石を送れなくなったなら、あとはゴールまでの手数が少ない方が勝ち
- しかしこの「相手陣地に石が送れない」が予想よりかなり厳しい条件だった
    - 石が2つの異なるマス目にある場合、6のマスで合流して2個になるようにすることで相手陣地に石を送ることが可能である
:
| 1 | 2 | 3 | 4 | 5 | 6 | 7 | op |
| -- | -- | -- | -- | -- | -- | -- | -- |
|  |  |  |  | o | o |  |  |
| 1 | 2 | 3 | 4 | 5 | 6 | 7 |  |
|  |  |  |  |  | oo |  |  |
| 1 | 2 | 3 | 4 | 5 | 6 | 7 |  |
|  |  |  |  |  |  | o | o |

なので実際の決着としては「相手がこちらの陣地に石を送るより先にこちらの石をなくせばOK」

- ゴール前に2以上9未満の数をキープすることが大事かも？
    - その状態なら1手で相手に石を送れる

- 6個の値の列について下記が決定論的に決まる
    - 全部なくなるまでの最短手数a
    - 相手に送り込むまでの最短手数b
- aは最大6
    - 相手からの送り込みが起きない限り単調に減少する
- bは無限大(送り込み不能)になりうる
    - 下手にプレイすると増える
    - 相手からの送り込みで減少する
- 自分の出番の後、相手のaが自分のb以外なら敗北
